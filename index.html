Dorian Gray's Metaphor Tracker

What is a metaphor tracker?

This project uses XML to create a website capable of detecting and analyzing metaphors and similes in The Picture of Dorian Gray by Oscar Wilde. The goal is to compile figurative language results and compare similarities between them — identifying recurring words, shared imagery, and patterns. This allows for clearer visualization of Wilde’s image system and stylistic tendencies.
